# Euphoric

#### The best CLI for TheAltening

##### Generate accounts with

```sh
$ generate [info=true]
```

##### Get more in depth info (Hypixel rank, levels, Skyblock coins, etc)

```sh
$ info <IGN/TOKEN>
```

##### View more commands with

```sh
$ help
```
